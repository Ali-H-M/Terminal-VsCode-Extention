# Create Your First Profile

A **profile** is a saved set of terminals that open together.

1. Click **New Profile** in the settings panel
2. Give it a name (e.g. *My Dev Setup*)
3. Add terminal commands — one per line (e.g. `npm run dev`)
4. Optionally add split panels for side-by-side terminals
5. Click **Save Profile**

> **Tip:** Enable **Auto-Launch** on a profile to open it automatically every time this workspace opens.
