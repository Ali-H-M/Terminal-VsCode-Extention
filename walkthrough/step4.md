# Share Profiles with Your Team

Commit a `.termprofile` file to your repo so everyone on the team gets the same terminal setup when they clone the project.

1. Run `Terminal Launcher: Create .termprofile for Workspace`
2. Select which profiles to include
3. Commit the generated `.termprofile` file to version control

When a teammate clones the repo and opens the workspace, Terminal Launcher detects the `.termprofile` and offers to import the profiles automatically.
